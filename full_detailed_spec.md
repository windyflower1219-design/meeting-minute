
# AI 회의록 생성 시스템 (HWPX 출력) - FULL 상세 명세서

---

# 1. 개요

본 시스템은 녹취록(TXT)과 회의자료(PPTX)를 입력받아
자동으로 **보고용 회의록(HWPX)**을 생성하는 시스템이다.

핵심 목적:
- 단순 요약 ❌
- 업무 실행 문서 생성 ⭕

---

# 2. 전체 처리 흐름

```
TXT + PPTX
→ 파싱
→ 세그먼트 분리
→ 안건 생성
→ 발언 정렬
→ 사실 추출
→ 의도 분류
→ Action Item 생성
→ 회의록 구성
→ 검증
→ HWPX 생성
```

---

# 3. 입력 명세

## 3.1 TXT

지원 형식:
- 화자 시간
- 화자: 내용

예:
최동혁 00:00
회의 시작하겠습니다.

## 3.2 PPTX

- Title → 안건
- Bullet → 세부 내용

---

# 4. 핵심 알고리즘

## 4.1 세그먼트 분리

기준:
1. 화자 변경
2. 의미 변경
3. 길이 제한 (1500 tokens)

---

## 4.2 중요도 계산

```
대표 발언 +3
지시문 포함 +2
기한 포함 +2
```

---

## 4.3 Action Item 생성

조건:
- 해야 / 확인 / 보고
- 담당자 추정 가능
- 작업 정의 가능

출력:
```
Task
Owner
Deadline
```

---

# 5. LLM 사용 전략

1차: 세그먼트 분석
2차: 안건 통합
3차: 문장 생성
